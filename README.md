# Cosmic-Flip
Cosmic Flip is a futuristic neon-themed memory card game built using HTML, CSS, and JavaScript. It features multiple categories, difficulty levels, cinematic animations, sound effects, countdowns, pause/resume system, responsive design, and immersive arcade-style gameplay.

🎮 Features
  - 🌌 Futuristic neon UI
  - 🃏 Smooth 3D card flip animations
  - 🎵 Background music & sound effects
  - ⏳ Countdown before gameplay
  - ⚡ Multiple difficulty levels
  - 🎯 Progress tracking system
  - 🏆 Best score saving using LocalStorage
  - 🎉 Confetti & particle effects
  - ⏸ Pause / Resume gameplay
  - 🏠 Home navigation system
  - 📱 Fully responsive mobile-friendly design
  - ✨ Cinematic glow effects & transitions
